# AXI-MM Based Non-Blocking Cache

A synthesizable, 4-way set-associative, non-blocking L1 cache with an AXI4 main-memory interface, implemented in SystemVerilog. The design supports hit-under-miss through a Miss Status Holding Register (MSHR) array, a fill-response FIFO, and a write buffer with RAW forwarding.

The full design report is available at [report.pdf](report.pdf).
